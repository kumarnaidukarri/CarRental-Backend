
tools :-
    Node.js, Express.js, Mongodb, mongoose 
    cors, dotenv, jsonwebtoken, bcrypt
    nodemon

    "Multer": 
      a npm package used to handle 'file uploads'.
      it provides a middleware function that will handle 'upload files'.
      it save inside "uploads" folder in 'Localdisk of our server' (or) 'Ram buffer memory'. 
      later, we can send file to cloud server(imagekit, aws s3). 

    "@Imagekit/Nodejs": 
       SDK package used to upload your mentioned files to the "Imagekit Cloud Server".  
       we can access them using CDNs.
